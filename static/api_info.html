API Information

Access the api at /api/CAMPUS/DAY/START/END?buildings=BUILDINGS&subjects=SUBJECTS

CAMPUS:
CAC = College Ave
BUS = Busch
LIV = Livingston
CD = Cook/Douglas
DAY:
M = Monday
T = Tuesday
W = Wednesday
TH = Thursday
F = Friday
S = Saturday
START/END
4 digit number representing a time between 0800 and 2230
BUILDINGS (optional)
comma separated list of 3 character building codes
SUBJECTS (optional)
comma separated list of 3 digit department codes
