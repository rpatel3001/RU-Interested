API Information

Get classes from /api/classes/CAMPUS/DAY/START/END?buildings=BUILDINGS&departments=DEPARTMENTS

CAMPUS:
CAC = College Ave
BUS = Busch
LIV = Livingston
CD = Cook/Douglas
DAY:
M = Monday
T = Tuesday
W = Wednesday
TH = Thursday
F = Friday
S = Saturday
START/END
4 digit number representing a 24-hour time in the format HHMM
BUILDINGS (optional)
comma separated list of 2 or 3 character building codes
full list found at /api/buildings
DEPARTMENTS (optional)
comma separated list of 3 digit department codes
full list found at /api/departments
